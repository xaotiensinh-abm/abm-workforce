# Goal

> **Use this skill when:** converting DOCX files to Markdown
Chuyển đổi các tệp .docx sang .md một cách chính xác, giữ nguyên cấu trúc (Heading, List, Table) để sử dụng cho các pipeline xử lý nội dung AI.

# Instructions
1. Xác định đường dẫn tệp nguồn (.docx).
2. Gọi script `scripts/convert_docx_to_md.py` bằng lệnh:
   ```powershell
   uv run --with mammoth --with markdownify "C:\Users\PC\.gemini\antigravity\skills\docx-to-md\scripts\convert_docx_to_md.py" --input <path_to_docx> --output <path_to_md>
   ```
3. Nếu không cung cấp `--output`, script sẽ tự động tạo một file .md cùng tên trong cùng thư mục.
4. Kiểm tra kết quả Markdown sau khi chuyển đổi.
