# 📘 Template Bảng Chủ Đề Phụ

> Copy bảng này khi khởi tạo module mới. Thay đổi cột "Chủ đề phụ" cho phù hợp.

## Cách sử dụng
1. Copy toàn bộ bảng bên dưới
2. Thay "Chủ đề phụ" bằng nội dung phù hợp module
3. Dán vào prompt Phase 1 cho mỗi lớp

---

## Bảng chủ đề phụ CHUNG (áp dụng được cho hầu hết module)

| Cấp | Lớp | Chủ đề phụ gợi ý |
|-----|-----|-------------------|
| Cấp 1 | 1 | Khái niệm cơ bản nhất, nhận biết đơn giản |
| | 2 | Kỹ năng phản xạ đầu tiên (nói KHÔNG, bước đi) |
| | 3 | Nhận diện tín hiệu nguy hiểm, phân biệt an toàn / không an toàn |
| | 4 | Hệ thống phân loại / cảnh báo (Đèn Xanh / Vàng / Đỏ) |
| | 5 | Kỹ năng tổng hợp nâng cao cho cấp 1 (xử lý tình huống phức) |
| Cấp 2 | 6 | Khía cạnh thể chất & tinh thần của chủ đề |
| | 7 | An toàn trên không gian mạng liên quan đến chủ đề |
| | 8 | Tình huống phức tạp: Grooming, Manipulation nâng cao |
| | 9 | Pháp lý, quyền trẻ em, Tổng đài 111, hệ thống hỗ trợ |
| Cấp 3 | 10 | Phân tích đa chiều, tư duy phản biện |
| | 11 | Trách nhiệm xã hội, nhóm yếu thế, cộng đồng |
| | 12 | Lãnh đạo, xây dựng văn hóa an toàn, chuẩn bị tự lập |
