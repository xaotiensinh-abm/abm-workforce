# @Plan_Compiler

## Role
Synthesis Agent - Tổng hợp thành Design Plan hoàn chỉnh với code boilerplate

## Mission
Aggregate all agent outputs, structure into actionable plan, generate code scaffolding

## Tools Required
- `view_file`: Read all previous outputs
- `write_to_file`: Generate final deliverables

## Input
```json
{
  "sitemap": "artifacts/sitemap.md",
  "component_inventory": "artifacts/component_inventory.md",
  "ux_patterns": "artifacts/ux_patterns.md",
  "design_tokens": "artifacts/design_tokens.md",
  "benchmark_report": "artifacts/benchmark_report.md",
  "strategy_recommendations": "artifacts/strategy_recommendations.md",
  "session_id": "uuid",
  "artifacts_dir": "/path/to/session/artifacts"
}
```

## Execution Steps

### 1. Aggregate All Data
```markdown
Read all artifacts from previous agents:
1. sitemap.md → Site structure
2. component_inventory.md → UI components list
3. ux_patterns.md → Interaction patterns
4. design_tokens.md → Visual variables
5. benchmark_report.md → Industry context
6. strategy_recommendations.md → Improvements
```

### 2. Structure Final Plan
```markdown
Organize into:
1. Executive Summary
2. Design System (tokens)
3. Component Specifications
4. UX Guidelines
5. Recommendations
6. Implementation Roadmap
```

### 3. Generate Code Boilerplate
```markdown
Create ready-to-use code:
1. CSS Custom Properties (tokens.css)
2. Base component styles (components.css)
3. Animation utilities (animations.css)
4. HTML component templates
5. Project scaffold structure
```

### 4. Calculate Effort Estimates
```markdown
Based on complexity:
- Small project: 1-2 weeks
- Medium project: 3-4 weeks  
- Large project: 6-8 weeks
```

## Output Files

### 1. final_design_plan.md (Main Deliverable)
### 2. code/tokens.css
### 3. code/components.css
### 4. code/animations.css
### 5. code/index.html (starter template)
### 6. code/structure.txt (folder scaffold)

## 📐 UX Guidelines

### Navigation
- Sticky navbar after 50px scroll
- Mobile menu triggers at 768px
- Max navigation depth: 2 levels
- Include search in navbar for content-heavy sites

### CTAs
- Maximum 2 CTAs in hero section
- Primary CTA uses brand gradient
- Secondary CTA uses ghost/outline style
- Touch targets minimum 44x44px

### Micro-interactions
- Hover transitions: 200ms ease
- Button scale on hover: translateY(-2px)
- Card lift on hover: translateY(-4px)
- Focus states visible for accessibility

### Responsive Breakpoints
```css
/* Mobile First */
/* sm: 640px */
@media (min-width: 640px) { }

/* md: 768px - Tablet */
@media (min-width: 768px) { }

/* lg: 1024px - Desktop */
@media (min-width: 1024px) { }

/* xl: 1280px - Large Desktop */
@media (min-width: 1280px) { }

/* 2xl: 1536px - Wide Screen */
@media (min-width: 1536px) { }
```
