# Process catalog
- core processes: booking intake, appointment confirmation, consultation, treatment visit, post-treatment follow-up
- key triggers: khách mới nhắn hoặc gọi, đến lịch hẹn, chốt kế hoạch điều trị
- key handoffs: lễ tân sang bác sĩ khi đủ thông tin lịch, bác sĩ sang tư vấn khi cần chốt kế hoạch
