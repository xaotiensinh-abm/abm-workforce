# Mode detection

## Build
Dùng khi user muốn dựng twin từ tài liệu mới.
Các cụm gợi ý:
- build twin
- dựng digital twin
- đọc bộ tài liệu này và đóng gói cách team vận hành
- tạo twin pack cho team hoặc quy trình

→ **Khi xác nhận mode build**: đọc `references/mode-build-instructions.md` để lấy checklist 11 bước.

## Operate
Dùng khi user đã có twin hoặc muốn tạo artefact mới từ twin.
Các cụm gợi ý:
- dùng twin này để tạo SOP
- tạo onboarding guide theo twin
- chuẩn hóa report template theo twin

→ **Khi xác nhận mode operate**: đọc `references/mode-operate-instructions.md` để lấy checklist 7 bước.

## Update
Dùng khi user muốn đưa tài liệu mới vào twin hiện có.
Các cụm gợi ý:
- update twin
- bổ sung tài liệu mới
- refresh twin theo policy mới

→ **Khi xác nhận mode update**: đọc `references/mode-update-instructions.md` để lấy checklist 5 bước.

## Audit
Dùng khi user muốn kiểm tra độ đúng hoặc độ phủ.
Các cụm gợi ý:
- audit theo twin
- tài liệu này có bám twin không
- kiểm tra gap hoặc drift

→ **Khi xác nhận mode audit**: đọc `references/mode-audit-instructions.md` để lấy checklist 6 bước.
