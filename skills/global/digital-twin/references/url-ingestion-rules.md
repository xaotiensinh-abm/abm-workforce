# URL ingestion rules

Chỉ dùng public URL đọc được nội dung chính.

## Cần giữ
- title
- author nếu có
- published date nếu có
- main content
- table hoặc bullet chứa process, owner, metric, rule

## Cần loại
- nav, footer, sidebar
- comments, related posts, widgets
- cookie banners, promotional rails

## Khi URL yếu
Nếu trang quá động, bị paywall, cần login, hoặc trích xuất rỗng:
- nói rõ không ingest được
- yêu cầu user cung cấp bản text hoặc file
