## Camera Angles (Góc máy)

```
        BIRD'S EYE (nhìn thẳng xuống)
              ↓ God-view, bản đồ, pattern
        ┌─────────────┐
        │  HIGH ANGLE  │ → vulnerability, nhỏ bé
        │      ↓       │
        │  EYE LEVEL   │ → neutral, bình thường
        │      ↓       │
        │  LOW ANGLE   │ → power, đe dọa, hùng vĩ
        └─────────────┘
              ↓
        WORM'S EYE (nhìn thẳng lên)
              → kinh sợ, choáng ngợp

        DUTCH ANGLE (nghiêng 15-30°)
              → bất an, thế giới nghiêng

        POV (first person)
              → immersive, nhập vai
```

## Transition Types (Scott McCloud)

```
1. MOMENT → MOMENT    ⏱️  Cùng hành động, micro-thời gian
   Ví dụ: Ngón tay tiến đến nút → bấm nút

2. ACTION → ACTION    🏃  Cùng nhân vật, khác hành động
   Ví dụ: Nhìn giá → bấm nhận → lên xe

3. SUBJECT → SUBJECT  👥  Cùng cảnh, khác chủ thể
   Ví dụ: Dũng trên xe → Bà lão ở gốc cây

4. SCENE → SCENE      🌃  Nhảy location hoặc time
   Ví dụ: Đường phố → Phòng trọ

5. ASPECT → ASPECT    🎭  Khác khía cạnh, cùng mood
   Ví dụ: Hồ đen + cây trụi + đèn nhòe = cô đơn

6. NON-SEQUITUR       ❓  Không liên hệ logic
   Ví dụ: Ký tự cổ ↔ Đường phố Hà Nội
```

## Panel Layout Cheat Sheet

```

                      Reveal, impact       └─────────────┘
                                           Zooming sequence

│  1   ╲  │           │ 1 │ 2 │ 3 │       ├───────────┤

│    2    │           │  MAIN(4) │       ├───────┤
```
