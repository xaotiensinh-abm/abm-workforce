# Prompt Templates — Manga 4-6 Panel (Tiếng Việt)

## Template 1: Trang thiết lập (Establishing) — 5 panels

```

Panel 2 (giữa trái, 15%): [CỠ CẢNH] [CHI TIẾT NHÂN VẬT]. Panel 3 (giữa phải, 15%): Đặc tả [VẬT THỂ/CHI TIẾT QUAN TRỌNG]. Panel 4 (dưới trái, 20%): Trung cảnh [NHÂN VẬT TRONG BỐI CẢNH]. ## Style Lock Presets

### Neo-Noir Urban (tu tiên đô thị)
```
Phong cách manga noir tối, ink wash, nét vẽ đậm tương phản cao,
cảnh đường phố Việt Nam ướt mưa. Kỹ thuật ink wash với accent
chọn lọc (amber đèn đường, xanh lục năng lượng huyền bí). Nét
chi tiết, chiều sâu không gian, bố cục cinematic. Chất lượng
manhwa Hàn kết hợp màu sắc manhua Trung.
```

### Classic Manga B&W
```
Phong cách manga đen trắng truyền thống. Nét vẽ sạch chính xác,
screentone shading chi tiết, bố cục dynamic. Speed lines và
impact frames cho cảnh hành động. Khuôn mặt biểu cảm mạnh.
Thẩm mỹ manga Nhật Bản.
```

### Colorful Manhua (xianxia)
```
Phong cách manhua Trung Quốc full color rực rỡ. Màu bão hòa đậm,
thiết kế trang phục chi tiết với áo dài hoa văn. Hiệu ứng khí/
linh lực phát sáng gradient. Nền vẽ lush painted. Tỉ lệ nhân vật
realistic nhưng lý tưởng hóa.
```

### Webtoon Pastel (romance/slice-of-life)
```
Phong cách webtoon pastel nhẹ nhàng. Nét vẽ minimal sạch sẽ,
đổ bóng dịu. Ánh sáng ấm ambient. Biểu cảm chibi cho khoảnh
khắc hài hước. Hiệu ứng hoa/lấp lánh overlay cho cảm xúc.
Nền đơn giản chi tiết chọn lọc. Không khí ấm áp.
```
