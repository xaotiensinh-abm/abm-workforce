# 12-Agent Audit Framework — Chi tiết đánh giá

## Mục đích

Hệ thống đánh giá đa chiều 12 agent được thiết kế để phát hiện
lỗi mà đánh giá đơn chiều bỏ sót. Mỗi agent có "perspective" riêng,
tạo ra adversarial coverage.

---

## Bảng Agent chi tiết

### 🎯 Agent 1: Content Accuracy (Trọng số 10%)

**Nhiệm vụ:** Kiểm tra nội dung chuyên môn đúng, đủ, không mâu thuẫn.

**Checklist:**
- [ ] Mọi claim chuyên môn có citation (năm + nguồn)
- [ ] Số liệu không mâu thuẫn giữa các slide
- [ ] Framework/acronym nhất quán (nếu nói "4 bước" → đếm đúng 4)
- [ ] Case study có thật, đúng năm, đúng tổ chức
- [ ] Learning objectives khớp với nội dung thực tế

### 🎓 Agent 2: Instruction Design (10%)

**Nhiệm vụ:** Đánh giá theo Gagné 9 Events và Bloom Taxonomy.

**Checklist:**
- [ ] Hook (Event 1) trong 5 phút đầu
- [ ] Objectives stated (Event 2) rõ ràng
- [ ] Recall prior (Event 3) — recap bài trước
- [ ] Guided practice (Event 6) — ít nhất 1 bài thực hành
- [ ] Assessment (Event 8) — quiz/poll/exercise
- [ ] Retention (Event 9) — cheat sheet/takeaway

### 🎙️ Agent 3: Audio/Production Quality (8%)

**Nhiệm vụ:** Đảm bảo script sẵn sàng cho voice recording.

**Checklist:**
- [ ] Breath marks mỗi 100-120 từ
- [ ] SFX cues đồng nhất (ding, transition)
- [ ] TGĐ endorsement slot (S0)
- [ ] Demo backup cho 100% demo slots
- [ ] Word count khớp timing @140w/m
- [ ] Buffer 10-20% (cảnh báo nếu >20%)

### 🏭 Agent 4: Domain Expert (10%)

**Nhiệm vụ:** Thuật ngữ ngành, ví dụ thực tế, case study phù hợp.

**Checklist:**
- [ ] Thuật ngữ ngành chính xác
- [ ] Ví dụ phù hợp với công việc hàng ngày của đối tượng
- [ ] Case study relevant (cùng ngành hoặc có bài học rõ)
- [ ] Analogies ngành (vận hành CDU, reactor, SOP...)

### 🧪 Agent 5: QA Consistency (10%)

**Nhiệm vụ:** Tìm mâu thuẫn, lỗi logic, numbering sai.

**Checklist:**
- [ ] Heading numbering khớp nội dung
- [ ] Cross-reference bài trước/sau đúng
- [ ] Slide numbering liên tục, không nhảy
- [ ] Framework description khớp (nếu nói "6 bước" → liệt kê 6)
- [ ] Không typo, không duplicate breath marks

### 🧠 Agent 6: Psychology/Engagement (8%)

**Checklist:**
- [ ] Hook tạo emotional response (sợ, tò mò, bất ngờ)
- [ ] Emotion arc: hook → tension → resolution → empowerment
- [ ] Analogies relatable cho đối tượng
- [ ] Pacing variation (không monotone)

### 🏢 Agent 7: L&D Director (8%)

**Checklist:**
- [ ] Actionable takeaway (cheat sheet, checklist)
- [ ] Assessment rubric cho exercises
- [ ] Follow-up mechanism (khảo sát, quiz sau 2 tuần)
- [ ] Transfer plan (áp dụng vào công việc thật)

### 🔄 Agent 8: Change Management (8%)

**Checklist:**
- [ ] Tone "enabling" không "threatening"
- [ ] Address resistance ("AI không thay thế...")
- [ ] Org-level change seeds (thư viện prompt, champions)
- [ ] Success stories trước fear stories

### ⚖️ Agent 9: Legal/Compliance (8%)

**Checklist:**
- [ ] Citations đủ cho mọi claim pháp lý
- [ ] Regulatory references đúng (NĐ, Luật, ISO)
- [ ] Disclaimer nếu cần
- [ ] Data handling rules reinforced

### 📺 Agent 10: Production Feasibility (8%)

**Checklist:**
- [ ] Demo slot feasible (software available?)
- [ ] Backup plan cho mọi demo
- [ ] Slide count phù hợp timeline
- [ ] Tech requirements listed

### ♿ Agent 11: Accessibility (6%)

**Checklist:**
- [ ] English terms explained inline
- [ ] Jargon minimized hoặc defined
- [ ] Content accessible cho mọi level

### 🇻🇳 Agent 12: VN Culture (6%)

**Checklist:**
- [ ] Kính ngữ phù hợp ("anh chị")
- [ ] Cấu trúc tổ chức VN (Đảng ủy, Ban TGĐ)
- [ ] Corporate hierarchy respected (TGĐ endorsement)
- [ ] Ví dụ localize (Vietnamese business context)
