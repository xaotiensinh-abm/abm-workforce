# Design Pipeline Workflow

## Step 1: Brief
1. Xác định target audience
2. Collect brand guidelines (nếu có)
3. Reference images / inspiration
4. Technical constraints (platform, size)

## Step 2: Design
1. Color palette + typography selection
2. Layout wireframe (mobile-first)
3. Component design (spawn UISubAgent if complex)
4. Visual assets (spawn ImageSubAgent if needed)

## Step 3: Build
1. Code HTML/CSS/JS based on design
2. Responsive testing (mobile/tablet/desktop)
3. Animation/interaction implementation
4. Accessibility check

## Step 4: Polish
1. Micro-animations, hover effects
2. Performance optimization
3. Cross-browser testing
4. Submit to W8:CriticAgent for review
