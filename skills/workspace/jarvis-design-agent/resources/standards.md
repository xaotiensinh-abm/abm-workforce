# Design Standards

## Color Palettes
- Dark mode: HSL-based, avoid pure black (#000)
- Glassmorphism: backdrop-blur + semi-transparent backgrounds
- Gradients: 2-3 colors max, smooth transitions
- Accessibility: contrast ratio ≥ 4.5:1 for text

## Typography
- Primary: Inter, Outfit, or Be Vietnam Pro
- Secondary: Merriweather, Playfair Display
- Scale: 16px base, 1.25 ratio (minor third)
- Line height: 1.5 for body, 1.2 for headings

## Layout
- Max width: 1200px container
- Grid: 12-column, responsive breakpoints
- Spacing: 4px/8px base unit system
- Mobile: 375px min, touch targets ≥ 44px

## Image Prompts (Nano Banana Pro)
- Formula: [SUBJECT]+[ACTION]+[LOCATION]+[CAMERA]+[LIGHTING]+[STYLE]
- Consistency: face/product lock FIRST in prompt
- No turning for I2V without back reference
