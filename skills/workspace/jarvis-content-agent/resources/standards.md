# Content Standards

## Writing Rules
- **Voice**: Professional nhưng gần gũi, không academic
- **Structure**: H1 → H2 → H3 hierarchy, bullet points cho scan
- **Length**: Blog 1500-2500 words, Social 50-300 chars, Email 200-500 words
- **SEO**: 1 primary keyword per piece, 2-3 secondary, density 1-3%

## Copywriting Frameworks
- **AIDA**: Attention → Interest → Desire → Action
- **PAS**: Problem → Agitate → Solution
- **BAB**: Before → After → Bridge

## Citation Format
- In-text: `[1]`, `[2]` sau claim
- References section cuối bài với full source + year

## Quality Checklist
- [ ] Có outline trước khi viết
- [ ] Target keyword xác định
- [ ] Meta description (155 chars)
- [ ] Internal links (2-3 per 1000 words)
- [ ] CTA rõ ràng
- [ ] Proofread: grammar, spelling, flow
