# Data Pipeline Workflow

## Step 1: Assess
1. Identify data sources and formats
2. Assess data quality and completeness
3. Plan extraction strategy

## Step 2: Extract (spawn ETLSubAgent if complex)
1. Parse files (PDF/DOCX/XLSX/CSV)
2. API calls for external data
3. Web scraping if needed (ethical)

## Step 3: Transform
1. Clean and normalize data
2. Validate accuracy
3. Enrich with additional sources

## Step 4: Load / Analyze
1. Structure for target system
2. Generate analysis/insights
3. Create reports/visualizations

## Step 5: Automate (spawn AutomationSubAgent if needed)
1. Set up recurring data pipelines
2. CRM/tool integrations
3. Alert/notification rules
