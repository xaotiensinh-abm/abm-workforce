# Git Workflow Optimization

## Overview

Tối ưu quy trình git push lên GitHub cho ABM-Vibecoding workspace.

**Nguyên tắc cốt lõi:** Push có hệ thống — clean commits, proper branching, verified code.

## Pre-Push Checklist

```
TRƯỚC KHI push BẤT KỲ code nào:

1. VERIFY  — Tests pass? Build ok? (fresh run)
2. CLEAN   — Git status clean? Staged đúng files?
3. COMMIT  — Message theo convention? Squash nếu cần?
4. BRANCH  — Đúng branch? Base branch updated?
5. REVIEW  — Self-review diff trước push?
6. PUSH    — Push lên đúng remote + branch
```
