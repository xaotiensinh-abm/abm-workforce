# 💡 Brainstorm — Ý tưởng → Thiết kế

## Bước 1: Load Brainstorming Skill
Đọc và thực thi toàn bộ skill:
- Load file: `{project-root}/.agents/skills/brainstorming/SKILL.md`
- Follow ALL steps trong Checklist của skill
- Tuân thủ HARD-GATE: KHÔNG code trước khi design approved

## Bước 2: Thực hiện Brainstorming Flow
1. Explore project context
2. Hỏi clarifying questions (1 câu/lần)
3. Propose 2-3 approaches
4. Present design → get CEO approval
5. Write design doc
6. Spec review loop

## Bước 3: Chuyển sang Planning
Sau khi design approved → invoke `/write-plan` để tạo implementation plan.
