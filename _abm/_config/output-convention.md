# ABM Output Convention

## Directory Structure
_abm-output/
  content/       Blog posts, articles
  social/        Social media posts
  email/         Email sequences, newsletters
  web/           Websites, landing pages
  reports/       Business reports, KPIs
  hr/            JDs, reviews, onboarding docs
  training/      Course materials, slides
  seo/           SEO audits, keyword maps
  proposals/     Client proposals
  slides/        Presentation decks

## File Naming
{date}_{type}_{topic}.{ext}
Example: 2026-03-14_blog_ai-coaching-roi.md
Example: 2026-03-14_email_welcome-sequence.md

## Version Control
filename_v1.md → filename_v2.md (major revision)
filename_v1.1.md (minor edit)
