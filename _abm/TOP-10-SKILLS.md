## Cách sử dụng

Bạn **KHÔNG CẦN** gọi tên skill. Chỉ cần nói yêu cầu:

```
"jarvis - phân tích cơ hội kinh doanh app quản lý gym"
→ Jarvis tự động chọn: startup-analyst + market-sizing-analysis

"jarvis - viết content plan cho blog tech"
→ Jarvis tự động chọn: seo-content-planner + content-creator

"jarvis - nghiên cứu thị trường AI tại Việt Nam"
→ Jarvis tự động chọn: deep-research + competitive-landscape
```

Jarvis tự phân tích yêu cầu → chọn skill phù hợp → thực hiện → xác minh → trả kết quả.
