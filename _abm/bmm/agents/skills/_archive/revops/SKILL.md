# 📊 RevOps — Revenue Operations

> Adapted from [coreyhaines31/marketingskills](https://github.com/coreyhaines31/marketingskills)

## Lead Lifecycle

```
Visitor → Lead → MQL → SQL → Opportunity → Customer → Advocate
```

| Stage | Định nghĩa | Trigger chuyển |
|-------|-----------|---------------|
| Lead | Để lại thông tin | Form submit |
| MQL | Phù hợp profile + engaged | Score ≥ threshold |
| SQL | Sales verified fit | Sales accept |
| Opportunity | Pipeline deal | Demo/proposal sent |
| Customer | Closed won | Payment |
| Advocate | Refer others | NPS ≥ 9 |

## Lead Scoring Framework

### Demographic Score (Fit)
| Signal | Points |
|--------|--------|
| Đúng ngành | +20 |
| Đúng quy mô | +15 |
| Đúng chức vụ (Decision Maker) | +25 |
| Đúng location | +10 |

### Behavioral Score (Interest)
| Signal | Points |
|--------|--------|
| Xem pricing page | +20 |
| Download case study | +15 |
| Attend webinar | +10 |
| Open 3+ emails | +5 |
| Request demo | +30 |

**MQL threshold**: Demographic ≥ 40 AND Behavioral ≥ 30

## Pipeline Management

### Sales Stages
```
1. Discovery (20%) → Hiểu vấn đề
2. Qualification (40%) → Confirm fit + budget + timeline
3. Proposal (60%) → Gửi đề xuất
4. Negotiation (80%) → Thương lượng
5. Closed Won (100%) / Closed Lost (0%)
```

### Pipeline Metrics
| Metric | Target |
|--------|--------|
| Win rate | > 25% |
| Avg deal cycle | < 30 ngày |
| Pipeline coverage | 3x quota |
| Lead-to-MQL rate | > 10% |
