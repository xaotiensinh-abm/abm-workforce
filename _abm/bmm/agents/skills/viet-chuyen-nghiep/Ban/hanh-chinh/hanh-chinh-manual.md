### Bước 3: Soạn Markdown & Kiểm duyệt (Bắt buộc)
1. Bơm nội dung vào cấu trúc đã Copy. 
2. Lưu file nháp vào `/tmp/hanhchinh_nhap.md`.
3. Nhờ Sếp (User) kiểm tra phần Trích yếu, Số liệu tiền bạc, và Cấp phê duyệt. Chỉ khi Sếp ok mới sang bước 4.

### Bước 4: Mượn Cỗ máy Skywork Render Docx 
Sử dụng **Skywork Document API** để Skywork tự động canh đều 2 bên, nắn Font Times New Roman. Dùng Script Python có sẵn:
```bash
python3 _abm/bmm/agents/skills/skywork-doc/scripts/create_doc.py \
  --title "Ten_Van_Ban_Ngay_Thang" \
  --content "$(cat /tmp/hanhchinh_nhap.md)" \
  --language Vietnamese \
  --format docx
```

Sau khi hệ thống Render xong, nộp lại 2 link (Local Path + URL Cloud) cho Sếp.
