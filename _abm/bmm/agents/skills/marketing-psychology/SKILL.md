# Marketing Psychology — Tâm lý học marketing & Mental Models

> Adapted from [coreyhaines31/marketingskills](https://github.com/coreyhaines31/marketingskills)

## Cách sử dụng
Đây là thư viện mental models. Khi viết copy, thiết kế pricing, hoặc CRO — tra cứu model phù hợp và áp dụng.

## Foundational Thinking Models

| Model | Áp dụng |
|-------|---------|
| **First Principles** | Phân tích vấn đề từ gốc, không copy đối thủ |
| **Jobs to Be Done** | Khách hàng "thuê" sản phẩm để hoàn thành "job" gì? |
| **Pareto 80/20** | 20% nỗ lực → 80% kết quả. Focus vào đâu? |
| **Inversion** | Muốn thành công → hỏi "cách nào để thất bại?" rồi tránh |
| **Second-Order Thinking** | Hậu quả bậc 2 của quyết định này là gì? |
| **Opportunity Cost** | Làm A nghĩa là không làm B. B có giá trị hơn? |

## Tâm lý người mua

| Bias/Effect | Ứng dụng |
|-------------|----------|
| **Loss Aversion** | "Đừng để mất 30% khách" > "Tăng 30% khách" |
| **Social Proof** | Testimonials, logo khách hàng, số liệu |
| **Scarcity** | "Còn 5 suất" — khan hiếm tạo hành động |
| **Anchoring** | Hiện giá cao trước → giá thật cảm thấy rẻ |
| **Endowment Effect** | Free trial → cảm giác "đã sở hữu" → khó bỏ |
| **Mere Exposure** | Thấy thương hiệu nhiều lần → trust tăng |
| **Bandwagon** | "10,000 doanh nghiệp đã dùng" |
| **Zero-Price Effect** | "Miễn phí" có sức mạnh phi tuyến tính |
| **Peak-End Rule** | Trải nghiệm cuối cùng quyết định ấn tượng |

## Pricing Psychology

| Technique | Rule |
|-----------|------|
| **Charm Pricing** | 99,000₫ < 100,000₫ (left-digit effect) |
| **Good-Better-Best** | 3 gói — middle gói bán chạy nhất |
| **Rule of 100** | < 100: dùng % off. > 100: dùng số tiền off |
| **Decoy Effect** | Gói "decoy" làm gói target hấp dẫn hơn |

## Persuasion Models

| Model | Cách dùng |
|-------|----------|
| **AIDA** | Attention → Interest → Desire → Action |
| **Reciprocity** | Cho giá trị free → họ muốn "trả lại" |
| **Authority** | Expert endorsement, certifications, awards |
| **Commitment** | Micro-commitment nhỏ → commitment lớn hơn |
| **Foot-in-Door** | Xin nhỏ trước → xin lớn sau |
| **Framing** | Cùng data, frame khác nhau → cảm nhận khác |

## Quick Reference

Khi viết copy → Check:
- [ ] Dùng Loss Aversion chưa?
- [ ] Có Social Proof không?
- [ ] CTA có Scarcity/Urgency không?
- [ ] Pricing dùng Anchoring chưa?
- [ ] First section có hook (Curiosity Gap) không?
