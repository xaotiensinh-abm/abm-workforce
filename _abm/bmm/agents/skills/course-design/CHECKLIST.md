# Course Design — Checklist & Examples

## Quality Checklist
- [ ] Learning objectives clear (SMART format)
- [ ] Content structured (Introduction > Core > Practice > Assessment)
- [ ] Assessment aligned with objectives
- [ ] Multimedia elements included
- [ ] Accessibility standards met
- [ ] Feedback mechanisms in place

## Example: AI Coaching Course Structure
```yaml
course: "ABM Workforce Mastery"
duration: "12 weeks"
modules:
  1: "AI Fundamentals" (week 1-2)
  2: "Prompt Engineering" (week 3-4)
  3: "Custom Bots" (week 5-6)
  4: "Automation" (week 7-8)
  5: "Content AI" (week 9-10)
  6: "Scale & Optimize" (week 11-12)
```
