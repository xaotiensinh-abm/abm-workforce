# Goal
Sáng tác và sản xuất âm nhạc chuyên nghiệp hoàn toàn tự động thông qua Mureka AI API. Bao gồm sáng tác lời (Lyrics), tạo nhịp (Beat/Instrumental), thu thanh và hoà âm (Song with Vocals), hoặc clone giọng hát thật. Hỗ trợ đa ngôn ngữ.

# Instructions

## Bước 1: Giao Tiếp & Nhận Yêu Cầu
Agent phân tích câu văn tự nhiên của User (VD: "Làm bài nhạc pop buồn, có đoạn cao trào cuối") ra thành các chỉ số (Parameters):
- **genres**: Pop, Rock, Guofeng...
- **moods**: Happy, Sad, Energetic...
- **instruments**: Piano, Guitar, Drums...
- **vocals**: Female, Male, Husky, Clear (hoặc Instrumental only).
- **bpm**: 120, slow, uptempo.

## Bước 2: Tạo Lời (Lyrics)
Nếu User yêu cầu bài hát có giọng hát mà CHƯA CÓ lời bài hát, Agent PHẢI chủ động gọi Lệnh tạo Lời trước:
```bash
python3 _abm/bmm/agents/skills/skywork-music-maker/scripts/mureka.py lyrics generate "Mô tả bài hát..."
```
*(Hoặc Agent tự dùng System LLM để tự viết lời, nhớ kẹp tag `[Verse]`, `[Chorus]`, `[Bridge]` để model dễ hát).*

## Bước 3: Tạo Âm Nhạc
Khi đã có đủ `prompt` thông số tiếng Anh + `lyrics` (nếu cần), Agent gọi Script sinh nhạc. Tất cả lệnh xuất ra mặc định đẩy vào CLI chạy. Nếu nhạc nền (không cần lời), dùng lệnh `instrumental`.

```bash
# Tạo Bài Hát Có Lời
python3 _abm/bmm/agents/skills/skywork-music-maker/scripts/mureka.py song \
  --lyrics "[Verse]\nHello world\n[Chorus]\nSing with me" \
  --prompt "indie pop, warm, 110 BPM, acoustic guitar, soft drums, male vocal" \
  --output ./output_music_folder

# Tạo Nhạc Nền (Beat / BGM)
python3 _abm/bmm/agents/skills/skywork-music-maker/scripts/mureka.py instrumental \
  --prompt "ambient electronic, calm, 80 BPM, soft pads, no percussion, background music" \
  --output ./output_bgm_folder
```
