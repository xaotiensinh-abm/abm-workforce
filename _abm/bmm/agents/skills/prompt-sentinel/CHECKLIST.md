# Prompt Sentinel — Checklist

## Detection Checklist
- [ ] Injection attempt identified
- [ ] Role play manipulation blocked
- [ ] Data exfiltration prevented
- [ ] Instruction override rejected
- [ ] Context window poisoning caught

## Response Protocol
1. DETECT: Pattern match against known attack vectors
2. CLASSIFY: Low/Medium/High/Critical severity
3. BLOCK: Reject malicious prompt
4. LOG: Record attempt for analysis
5. ALERT: Notify if critical
