# HR Operations — Checklist

## Recruitment Checklist
- [ ] JD approved by manager
- [ ] Salary range confirmed
- [ ] Job board postings live
- [ ] Screening criteria defined

## Onboarding Checklist
- [ ] Welcome kit prepared
- [ ] IT access provisioned
- [ ] Buddy assigned
- [ ] 30-60-90 day plan created

## Offboarding Checklist
- [ ] Exit interview scheduled
- [ ] Access revoked
- [ ] Equipment returned
- [ ] Knowledge transfer completed
