# 🔄 Programmatic SEO — Scaled Page Generation

## Sử dụng khi

- Cần tạo hàng trăm/ngàn landing pages từ data
- Location pages (dịch vụ tại [city])
- Comparison pages (X vs Y)
- Integration pages (works with [tool])
- Template/example pages

## KHÔNG sử dụng khi

- Content marketing thông thường → dùng `content-strategy`
- Technical SEO → dùng `seo-audit`
- AI SEO → dùng `ai-seo`
- Thin content / spam pages → TUYỆT ĐỐI KHÔNG

## TEMPLATES PHỔ BIẾN

| Template | Ví dụ | Volume |
|----------|-------|:------:|
| [Product] cho [Industry] | "CRM cho bất động sản" | 50-200 |
| [Product] tại [City] | "Dịch vụ SEO tại Hà Nội" | 50-100 |
| [Product] vs [Competitor] | "Notion vs Trello" | 20-50 |
| [Product] + [Integration] | "Zapier + Slack integration" | 30-100 |
| [Template] mẫu [Type] | "Hợp đồng mẫu thuê nhà" | 50-200 |

## WORKFLOW

```
1. RESEARCH — Tìm data set (cities, tools, industries)
2. TEMPLATE — Tạo page template với variables
3. GENERATE — Fill template × data = pages
4. UNIQUE — Thêm unique content mỗi page (30%+ unique)
5. LINK — Internal linking giữa pages
6. SUBMIT — Sitemap + Google Search Console
7. MONITOR — Index rate, traffic, rankings
```

## QUALITY CHECKLIST

- [ ] Mỗi page ≥ 30% unique content (không duplicate)
- [ ] Title tag unique cho mỗi page
- [ ] Meta description unique
- [ ] Internal links đến related pages
- [ ] Schema markup (FAQ, How-to)
- [ ] Canonical tags đúng
- [ ] No-index thin pages (< 300 words)

## QUY TẮC

1. **Quality trên hết** — không tạo thin content/doorway pages
2. Mỗi page phải **có giá trị thật** cho user
3. Monitor index rate — Google không index = quality issue
4. Start small (50 pages), measure, then scale
