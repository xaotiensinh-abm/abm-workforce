# 🤝 Sales Enablement — Hỗ Trợ Bán Hàng

> Adapted from [coreyhaines31/marketingskills](https://github.com/coreyhaines31/marketingskills)

## Tài liệu sales cần có

### 1. Sales Deck (10-15 slides)
```
1. Hook — vấn đề thị trường
2. Nỗi đau cụ thể (3 bullet)
3. Giải pháp hiện tại thiếu gì
4. Giới thiệu sản phẩm (1 slide)
5. Cách hoạt động (3 bước)
6. Kết quả khách hàng (case study)
7. Social proof (logos, testimonials)
8. Pricing (tùy context)
9. CTA + next steps
```

### 2. One-Pager
- 1 trang duy nhất — gửi sau meeting hoặc forward nội bộ
- Gồm: vấn đề, giải pháp, 3 benefits, 1 case study, CTA

### 3. Xử lý phản đối (Battle Cards)

| Phản đối | Response Framework |
|----------|-------------------|
| "Quá đắt" | Giá trị > giá. So sánh chi phí KHÔNG dùng sản phẩm. |
| "Chưa cần ngay" | Trigger event + chi phí trì hoãn |
| "Đang dùng X" | Switching cost thấp + benefit gap cụ thể |
| "Cần hỏi sếp" | Cung cấp one-pager để forward + ROI calculator |
| "Chưa tin" | Case study + trial/guarantee |

### 4. Demo Script
```
1. Discovery (5p): Hỏi vấn đề CỤ THỂ của họ
2. Setup (2p): "Để tôi show cách chúng tôi giải quyết..."
3. Show (10p): Demo 3 use cases LIÊN QUAN đến vấn đề họ nêu
4. Social proof (2p): "Khách hàng X cũng gặp vấn đề tương tự..."
5. CTA (1p): Next step rõ ràng
```

### 5. Email Templates
- Follow-up after meeting
- Proposal/quote email
- Check-in (no reply)
- Closing email
