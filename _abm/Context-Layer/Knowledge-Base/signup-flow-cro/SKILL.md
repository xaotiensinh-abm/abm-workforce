# 📝 Signup Flow CRO — Tối Ưu Đăng Ký

## Sử dụng khi

- Tỷ lệ signup thấp (< 30% visitors → signup)
- Form đăng ký quá dài/phức tạp
- Drop-off cao giữa các bước
- Cần A/B test signup flow

## KHÔNG sử dụng khi

- Landing page CRO → dùng `page-cro`
- Form lead gen → dùng `form-cro`
- Onboarding sau signup → dùng `email-sequence`

## CHECKLIST

### Reduce Friction
- [ ] Chỉ hỏi email ban đầu (tên, phone = later)
- [ ] Social login (Google, Facebook) có được không?
- [ ] Không CAPTCHA (dùng honeypot thay thế)
- [ ] Auto-fill browser data
- [ ] Show/hide password toggle
- [ ] Real-time validation (không submit rồi mới báo lỗi)

### Increase Motivation
- [ ] Value proposition rõ trên form
- [ ] Social proof gần CTA ("10,000+ users")
- [ ] Trust badges (SSL, money-back, privacy)
- [ ] Progress indicator (nếu multi-step)

### Quick Wins
```
✅ Giảm fields: 4 → 2 = +50% conversion (avg)
✅ Social login: +20-30% signup
✅ CTA text: "Bắt đầu miễn phí" > "Đăng ký"
✅ Remove navigation bar trên signup page
```

## QUY TẮC

1. Đo baseline TRƯỚC khi thay đổi
2. A/B test 1 biến mỗi lần
3. Mobile-first — 60%+ traffic từ mobile
