# 🛠️ Free Tool Strategy — Marketing Bằng Công Cụ Miễn Phí

## Sử dụng khi

- Cần lead gen channel bền vững (không paid ads)
- Muốn tạo tools/calculators miễn phí thu hút traffic
- SEO content không đủ — cần interactive content
- Build brand authority qua giá trị thật

## KHÔNG sử dụng khi

- Content marketing truyền thống → dùng `content-strategy`
- Landing page CRO → dùng `page-cro`
- Pricing → dùng `pricing-strategy`

## LOẠI TOOLS

| Loại | Ví dụ | Độ khó |
|------|-------|:------:|
| Calculator | ROI calculator, salary calculator | 🟢 |
| Generator | Name generator, email subject tester | 🟢 |
| Checker | SEO checker, password strength | 🟡 |
| Template | Invoice template, contract template | 🟢 |
| Analyzer | Website speed, competitor analyzer | 🟡 |
| Mini-app | Color palette, QR code generator | 🟡 |

## IDEATION FRAMEWORK

### Bắt đầu từ Pain Points
```
1. Khách hàng mục tiêu gặp vấn đề gì mỗi ngày?
2. Họ đang dùng Excel/Google Sheets cho gì?
3. Có task nào lặp đi lặp lại mà tool đơn giản giải quyết được?
4. Tool nào liên quan đến sản phẩm chính (adjacent)?
```

### Validate
- [ ] Có search volume cho "{X} calculator/tool/generator" không?
- [ ] Competitor đã có tool tương tự chưa?
- [ ] Build < 2 tuần?
- [ ] Liên quan sản phẩm chính?

## LEAD CAPTURE

```
Gating Options:
├── Ungated (100% free, SEO max) — cho simple tools
├── Partial gate (free basic, email for advanced) — recommended
├── Full gate (email required) — cho high-value tools
└── Progressive (free → upgrade for more) — cho SaaS
```

## QUY TẮC

1. Tool phải **giải quyết vấn đề thật** — không bait
2. **Adjacent** sản phẩm chính — không random
3. MVP trong 2 tuần, iterate sau
4. Measure: visitors → leads → customers
