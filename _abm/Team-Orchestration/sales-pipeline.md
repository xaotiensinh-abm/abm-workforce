# Sales Pipeline
## Flow: Lead → Qualify → Discovery → Trial → Close → Onboard

### Stage 1: Lead Capture (marketing-specialist)
- Sources: Content, events, referral, ads
- Output: Lead info in CRM / spreadsheet
- Skills: growth-engine, landing-page-builder

### Stage 2: Qualify (sales-specialist)
- Input: Lead info
- Check: ICP match, buying signals, disqualifiers
- Output: MQL → SQL conversion
- Skills: high-ticket-sales

### Stage 3: Discovery Call
- Input: SQL
- Format: SPIN framework, 30-45 min Zoom
- Output: Pain points mapped + next step
- Skills: high-ticket-sales

### Stage 4: Trial Session
- Input: Interested prospect
- Format: 90 min demo ABM Workforce
- Output: Wow experience + proposal

### Stage 5: Close
- Input: Proposal sent
- Techniques: Assumptive, Scarcity, Summary close
- Output: Signed contract + payment
- Skills: pricing-strategy, client-success

### Stage 6: Onboard (client-success)
- Input: New client
- Checklist: Welcome, NDA, Hub setup, Kickoff schedule
- Skills: client-success
