# Training Pipeline
## Flow: Design → Build → Deliver → Assess → Iterate

### Stage 1: Design (training-specialist)
- Input: Learning objectives + target audience
- Output: Course outline + module structure
- Skills: course-design

### Stage 2: Build (training-specialist + content-writer)
- Input: Course outline
- Output: Slide decks, handouts, exercises
- Skills: training-content, course-design

### Stage 3: Deliver (coaching session)
- Input: Built materials
- Format: Zoom 90min, GROW+A framework
- Output: Session recordings + recap
- Skills: workshop-facilitation

### Stage 4: Assess (training-specialist)
- Input: Session data + action items
- Output: Progress report + skill gaps
- Skills: student-assessment

### Stage 5: Iterate
- Input: Assessment results
- Action: Adjust next session, personalize
- Feed back to Stage 1 for next module
