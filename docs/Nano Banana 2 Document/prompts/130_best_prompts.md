## 🎯 Quick Reference

### Platform Compatibility

| Platform | How to Use |
|----------|------------|
| **Midjourney** | Copy entire prompt including `--ar`, `--style`, `--v`, `--stylize`, `--no` |
| **Stable Diffusion** | Use description as main prompt, move `--no` content to negative prompt field |
| **ChatGPT/DALL-E** | Use only the description text, skip `--` parameters |
| **Nano Banana Pro** | Copy entire prompt as-is |

### Parameter Guide

| Parameter | Purpose | Values |
|-----------|---------|--------|
| `--ar` | Aspect Ratio | `16:9` (YouTube), `1:1` (Instagram), `4:5` (Pinterest), `9:16` (TikTok) |
| `--style raw` | Hyper-realistic mode | `raw` only |
| `--v` | Model version | `7.0` (current) |
| `--stylize` | Creativity level | `150-200` (realistic) to `300-350` (artistic) |
| `--iw` | Image weight (IMG2IMG) | `1.5-2.0` (maintain features) |
| `--no` | Negative prompts | Exclude unwanted elements |

### Standard Negative Prompt
```
--no cartoon, anime, illustration, distorted faces, extra fingers, bad anatomy, blurry, low resolution, watermark, text, logo, oversaturated
```

## 💡 Advanced Tips

### 1. Composition Techniques
- Use photography terms: "rule of thirds", "low angle", "shallow depth of field"
- Specify lens: `35mm` (context), `50mm` (standard), `85mm` (portrait), `macro` (detail)

### 2. Lighting Descriptions
- **Golden hour**: Warm, nostalgic, hopeful
- **Rembrandt**: Dramatic portrait with triangle shadow
- **Studio 3-point**: Professional product photography
- **Cinematic**: Teal shadows, warm highlights

### 3. Style References
- Use "inspired by" or "aesthetic reminiscent of"
- Architecture references: "Bauhaus minimalism", "Art Deco geometry"
- Photography styles: "documentary", "editorial", "commercial"

### 4. IMG2IMG Best Practices
- Use well-lit, clear reference photos
- `--iw 1.8` maintains facial features
- Higher `--iw` = closer to original
- Better source images = better results
