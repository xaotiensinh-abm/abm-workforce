## Pro Tips: Portrait & Fashion
- IMG2IMG prompts require uploading reference photo first
- Use `--iw 1.6-1.8` to maintain facial features while transforming style
- Specify camera lens for different effects (35mm = context, 85mm = intimate)
- Include lighting style (Rembrandt, butterfly, split, rim)
- Reference specific aesthetic benchmarks (Vogue, Vanity Fair, Nike)
