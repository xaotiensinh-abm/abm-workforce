## 📖 Chi Tiết Từng Skill

### 1. `/review` — Đánh Giá Đa Chiều
**Đây là skill mạnh nhất.** 6 bước: Thu thập → Chấm điểm → Phản biện → Phát hiện → Đối chiếu → Hành động.

```
Ví dụ: /review đánh giá kế hoạch marketing Q2
```

### 2. `/marketing` — Content Factory
16 skills marketing tự động: copywriting, email, SEO, social, pricing.

```
Ví dụ: /marketing viết 5 bài Facebook cho sản phẩm mới
```

### 3. `/dev` — Phát Triển Phần Mềm
Bug fix, feature mới, refactor code. Có sprint planning + state machine.

```
Ví dụ: /dev fix bug login page
```

### 4-10: Tương tự — nói yêu cầu bằng tiếng Việt, Jarvis sẽ hướng dẫn.
